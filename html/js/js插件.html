# Toast提示

function toast(msg,duration){
	duration = isNaN(duration) ? 3000 : duration;	
	var div = document.createElement('div');
	div.innerHTML  = msg;
	div.style.cssText = `
		max-width:60%;min-width:150px;height:3rem;line-height:3rem;padding:0 14px;
		text-align: center;border-radius: 4px;
		font-size:1.3rem;color:rgb(255, 255, 255);background:rgba(0, 0, 0,.5);
		position:fixed;top:50%;left:50%;transform:translate(-50%,-50%),z-index:9999;
	`
	document.body.appendChild(div);
	setTimeout(function(){
		var d = 0.5;
		div.style.webkitTransition = "-webkit-transform " + d + "s ease-in, opacity " + d + "s ease-in";
		div.style.opacity = "0";
		setTimeout(function(){
			document.body.removeChild(div);
		},d*1000)
	},duration)
}
toast('Hello word')

# cssText

  • cssText的本质就是设置HTML元素的style属性
  • cssText的使用优势
    • 一般情况使用js设置元素对象样式会使用element.style.width="20px"这种方法。
    • 但是这样样式一多，代码就会重复，而且通过JS来覆写对象样式是比较典型的一种销毁原样式并重建的过程，这种销毁和重建都会增加浏览器的开销。
    • js中有cssText方法：
      	obj.style.cssText = "width:150px;height:3rem;line-height:3rem;"
    • 但是，这样会有一个问题，会把原有的cssText清掉，可以采用cssText累加的方法：
      	obj.style.cssText += "width:150px;height:3rem;line-height:3rem;"
	obj.style.cssText += ";width:150px;height:3rem;line-height:3rem;"//在IE中
